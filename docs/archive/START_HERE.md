## ⚡ Three Quick-Start Paths

### Path 1: Quick Test (Frontend Only)
**Time**: 30 seconds | **Requirements**: None

```bash
cd web
npm run dev
# → http://localhost:3000
```

**Use this to**: Test UI, check features, demo to others

---

### Path 2: Full Development (Backend + Frontend)
**Time**: 2 minutes | **Requirements**: Java 17+, Maven 3.9+

**Terminal 1:**
```bash
mvn spring-boot:run
# → http://localhost:8080/api/v1
```

**Terminal 2:**
```bash
cd web
npm run dev
# → http://localhost:3000
```

**Use this to**: Develop APIs, test backend logic, full-stack work

---

### Path 3: Production Simulation
**Time**: 1 minute | **Requirements**: None

```bash
cd web
npm run build
npm start
# → http://localhost:3000 (production mode)
```

**Use this to**: Test before deployment, check production behavior
